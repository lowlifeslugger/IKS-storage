Functions


A function is a group of statements that performs a specific task.


Types:

- Built-in functions — print(), sqrt(), len(), etc.

- User-defined functions — created by the programmer.


Advantages:

- Reusable — write once, use anywhere.

- Provides modularity to programs.

- Easy code maintenance.

- Reduces program length.

- Easier to locate and fix errors.


---


Function vs Method


Function Method
Written individually in a Python program Written inside a class
Called using its name: function_name() Called using an object: object.method()

Defining a Function


# Syntax
def function_name(parameters):
    """Docstring: optional description"""
    # function body (suite)
    statement(s)

- def keyword starts the definition.

- Parameter — a variable that receives data from outside the function.

- Colon : marks the beginning of the function body.

- Docstring — optional description of what the function does.

Calling a Function


A function runs only when it is called by its name.


# Example — sum of two numbers
def add(a, b):
    result = a + b
    print("Sum =", result)

# Call the function
add(5, 3)       # Output: Sum = 8
add(10, 20)     # Output: Sum = 30

# Dynamic typing — Python determines type at runtime
def add(a, b):
    print("Sum =", a + b)

add(5, 3)        # Sum = 8   (integers)
add(5.5, 3.2)    # Sum = 8.7 (floats)
add("Hello ", "World")  # Sum = Hello World (strings)

Returning Results from a Function


Use return to send a result back to the caller.


# Return a single value
def add(a, b):
    return a + b

result = add(5, 3)
print(result)     # 8

# Even or odd
def check_even_odd(n):
    if n % 2 == 0:
        return "Even"
    else:
        return "Odd"

print(check_even_odd(4))    # Even
print(check_even_odd(7))    # Odd

# Factorial
def factorial(n):
    fact = 1
    for i in range(1, n + 1):
        fact *= i
    return fact

for i in range(1, 11):
    print(f"{i}! = {factorial(i)}")
# Output:
# 1! = 1
# 2! = 2
# 3! = 6
# ...
# 10! = 3628800

# Prime check
def is_prime(n):
    x = 1
    for i in range(2, n):
        if n % i == 0:
            x = 0
            break
    return x

n = int(input("Enter number: "))
if is_prime(n):
    print(n, "is prime")
else:
    print(n, "is not prime")

# Trace for n=5:
# range(2,5) → i = 2, 3, 4
# 5%2≠0, 5%3≠0, 5%4≠0 → x stays 1 → return 1 → "5 is prime"

# Trace for n=4:
# range(2,4) → i = 2, 3
# 4%2==0 → x=0 → break → return 0 → "4 is not prime"

Returning Multiple Values


Python functions can return multiple values as a tuple.


# Return three values
def multi_return():
    a = 10
    b = 20
    c = 30
    return a, b, c     # returns tuple (10, 20, 30)

x, y, z = multi_return()
print(x, y, z)    # 10 20 30

# Arithmetic operations — return 4 values
def arithmetic(a, b):
    add = a + b
    sub = a - b
    mul = a * b
    div = a / b
    return add, sub, mul, div

a, s, m, d = arithmetic(10, 2)
print(f"Add={a}, Sub={s}, Mul={m}, Div={d}")
# Add=12, Sub=8, Mul=20, Div=5.0

Formal vs Actual Arguments


- Formal arguments — parameters listed in the function definition.

- Actual arguments — values passed during the function call.


def display(name, age):    # name, age = formal arguments
    print(name, age)

display("Alice", 25)       # "Alice", 25 = actual arguments

---


4 Types of Actual Arguments


1. Positional Arguments


Number and position must match exactly with the definition.


def display(name, age, city):
    print(f"Name: {name}, Age: {age}, City: {city}")

display("Alice", 25, "Goa")    # correct — matches order
# display(25, "Alice", "Goa")  # wrong result — wrong order

---


2. Keyword Arguments


Identified by parameter name — order doesn't matter.


def display(name, age, city):
    print(f"Name: {name}, Age: {age}, City: {city}")

display(age=25, city="Goa", name="Alice")    # order doesn't matter
# Output: Name: Alice, Age: 25, City: Goa

---


3. Default Arguments


A parameter assumes a default value if none is passed.


def greet(name, msg="Hello"):
    print(f"{msg}, {name}!")

greet("Alice")                  # Hello, Alice!   (uses default)
greet("Bob", "Good morning")    # Good morning, Bob!   (overrides default)

# Another example
def power(base, exp=2):
    return base ** exp

print(power(3))      # 9   (3²)
print(power(3, 3))   # 27  (3³)

---


4. Variable Length Arguments — `*args`


Accepts any number of positional arguments.


def total(*nums):
    s = 0
    for n in nums:
        s += n
    return s

print(total(1, 2, 3))           # 6
print(total(10, 20, 30, 40))    # 100
print(total(5))                 # 5

# Display all passed values
def show(*args):
    for val in args:
        print(val)

show("apple", "banana", "cherry")

---


4b. Keyword Variable Length Arguments — `**kwargs`


Accepts any number of keyword arguments as key-value pairs.


def show_info(**details):
    for key, value in details.items():
        print(f"{key}: {value}")

show_info(name="Alice", age=25, city="Goa")
# Output:
# name: Alice
# age: 25
# city: Goa

Local and Global Variables


Variable Scope Description
Local Inside the function only Declared inside a function; destroyed when function ends.
Global Entire program below declaration Declared outside all functions; accessible everywhere. 

x = 100    # global variable

def func():
    x = 50    # local variable (same name; SHADOWS global)
    print("Inside function:", x)    # 50

func()
print("Outside function:", x)    # 100 (global unchanged)

# When local and global have same name:
# → function uses LOCAL variable by default

Passing a Group of Elements to a Function


# Total and average of a group of numbers
def calc(nums):
    total = sum(nums)
    avg = total / len(nums)
    print("Total:", total, "  Average:", avg)

numbers = [10, 20, 30, 40, 50]
calc(numbers)
# Output: Total: 150   Average: 30.0

# Display a group of strings
def display_names(names):
    for name in names:
        print(name)

names = ["Alice", "Bob", "Charlie"]
display_names(names)

Lists


A list stores a group of elements that can be of different types. Lists are mutable — their contents can be changed.


> Lists are more versatile than arrays. Arrays store only one type; lists store any mix.


Creating Lists


# Empty list
lst = []

# List with mixed types
student = ["Alice", 20, 9.5, "Goa"]

# Using range()
lst1 = list(range(5))            # [0, 1, 2, 3, 4]
lst2 = list(range(2, 5))         # [2, 3, 4]
lst3 = list(range(5, 11, 2))     # [5, 7, 9]

Slicing Lists


Format: [start : stop : step]


x = [10, 20, 30, 40, 50]

x[1:3]       # [20, 30]
x[::2]       # [10, 30, 50]
x[::-1]      # [50, 40, 30, 20, 10]  — reverse
x[2:]        # [30, 40, 50]
x[:3]        # [10, 20, 30]

Accessing Elements


lst = [10, 20, 30, 40, 50]

# Using while loop
i = 0
while i < len(lst):
    print(lst[i])
    i += 1

# Using for loop
for item in lst:
    print(item)

# Display in reverse order
for i in range(len(lst) - 1, -1, -1):
    print(lst[i])

Updating Lists


lst = [10, 20, 30, 40, 50]

lst.append(60)             # [10, 20, 30, 40, 50, 60]
lst[1] = 200               # update single: [10, 200, 30, 40, 50, 60]
lst[1:3] = [20, 30]        # update range: [10, 20, 30, 40, 50, 60]
del lst[0]                 # delete first: [20, 30, 40, 50, 60]
lst.reverse()              # reverse in-place: [60, 50, 40, 30, 20]

List Operations


a = [1, 2, 3]
b = [4, 5, 6]

# Concatenation
c = a + b        # [1, 2, 3, 4, 5, 6]

# Repetition
d = a * 3        # [1, 2, 3, 1, 2, 3, 1, 2, 3]

List Methods


Method Description
append(x) Add x at the end
insert(i, x) Insert x at index i
remove(x) Remove first occurrence of x
pop(i) Remove and return element at index i
clear() Remove all elements
sort() Sort ascending in-place
sort(reverse=True) Sort descending in-place
reverse() Reverse in-place
index(x) Return index of first occurrence of x
count(x) Count occurrences of x
extend(lst) Append all elements of another iterable
copy() Return a shallow copy 

lst = [3, 1, 4, 1, 5, 9, 2]

print(max(lst))         # 9
print(min(lst))         # 1

lst.sort()
print(lst)              # [1, 1, 2, 3, 4, 5, 9]

lst.sort(reverse=True)
print(lst)              # [9, 5, 4, 3, 2, 1, 1]

Sorting Without sort() — Swap Method


lst = [5, 3, 8, 1, 4]
n = len(lst)

for i in range(n - 1):
    for j in range(n - 1 - i):
        if lst[j] > lst[j + 1]:
            lst[j], lst[j + 1] = lst[j + 1], lst[j]

print(lst)    # [1, 3, 4, 5, 8]

Tuples


A tuple stores a group of elements that can be of different types. Tuples are immutable — once created, elements cannot be changed.


> Use tuples to store data that must NOT be modified.


Creating Tuples


# Standard tuple
tp1 = (10, 20, 30)

# Without parentheses (also valid)
tp2 = 10, 20, 30

# Single-element tuple — MUST have trailing comma
tp3 = (42,)          # correct — this IS a tuple
tp4 = (42)           # WRONG  — this is just the integer 42

# Empty tuple
tp5 = ()

# From a list
tp6 = tuple([10, 20, 30])

Accessing Tuple Elements


tp = (10, 20, 30, 40, 50)

print(tp[0])        # 10    (first)
print(tp[-1])       # 50    (last)
print(tp[1:3])      # (20, 30)
print(tp[::2])      # (10, 30, 50)

# Using for loop
for item in tp:
    print(item)

# Using while loop
i = 0
while i < len(tp):
    print(tp[i])
    i += 1

Tuple Operations


t1 = (1, 2, 3)
t2 = (4, 5, 6)

len(t1)             # 3
t1 + t2             # (1, 2, 3, 4, 5, 6)   — concatenation
t1 * 3              # (1, 2, 3, 1, 2, 3, 1, 2, 3)  — repetition
3 in t1             # True   — membership check
7 not in t1         # True

# in and not in operators
print(10 in (10, 20, 30))      # True
print(50 not in (10, 20, 30))  # True

Tuple Immutability


tp = (10, 20, 30)
print(tp[0])         # 10

# tp[0] = 99         # TypeError: 'tuple' object does not support item assignment

# Tuples CANNOT use: append(), insert(), remove(), pop(), clear()
