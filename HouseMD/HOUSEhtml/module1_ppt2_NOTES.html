---


Central Processing Unit (CPU)


The CPU executes program instructions. In a microcomputer it is a single chip: the microprocessor.


a) Control Unit

- Directs movement of signals between memory and ALU.

- Directs control signals between CPU and I/O devices.

- Manages all computer resources.

- CPU instructions are built in as microcode — basic directions that tell the CPU how to execute complex operations.


b) Arithmetic Logic Unit (ALU)

- Arithmetic operations — addition, subtraction, multiplication, etc.

- Logical operations — comparisons: equal to, less than, greater than.

- Contains registers — high-speed memory locations built directly into the CPU that hold data currently being processed.

Memory


Primary Memory (directly accessible by CPU)


Type Nature Description
RAM (Random Access Memory) Volatile Temporary storage; loses data when powered off. Types: DRAM (Dynamic), SRAM (Static).
Cache Memory Volatile Smaller and faster than RAM; stores frequently accessed data. Levels: L1 (fastest/smallest), L2, L3.
ROM (Read-Only Memory) Non-volatile Stores firmware and boot instructions. Retains data when power is off.

Secondary Memory (non-volatile, long-term storage)


Access types:


Type Description Example
Sequential Access Must go through all preceding data to reach a point Magnetic tape
Random / Direct Access Any location accessed directly, in any order Hard disk, optical disk

Advantages of secondary storage: More economical, higher capacity, secure from unauthorized access.

Secondary Storage Devices


1. Magnetic Tapes

- Sequential access only.

- High capacity — can exceed other devices for archival use.

- Durable — last decades if stored properly.

- Cost-effective on a per-GB basis.

- Used for backups and archival.


2. Magnetic Disks


a) Hard Disks

- Thin steel platters with iron oxide coating.

- Multiple platters mounted on a vertical shaft rotating at high speed.

- Electromagnetic read/write heads on access arms.

- Data stored as magnetized spots on tracks.

- Each track stores thousands of bytes.

- Head crash — caused by dust particle making the read/write head contact the disk surface.

- Access time: 10 to 100 milliseconds.


Access time factors:


Factor Description
Seek Time Time to move the read/write head to the correct track
Rotational Delay Time for disk to rotate correct sector under the head
Data Transfer Time to transfer data to primary storage

b) Floppy Disks

- Soft magnetic disk; slower than hard disks.

- Lower capacity, less expensive, portable.


3. Optical Disks

- Data read/written by lasers. Large storage capacity.


Type Description
CD-ROM Data pre-recorded by vendor; read-only; permanent.
CD-R Compact Disc Recordable; write once, read many (WORM).
CD-RW CD-Rewritable; can write in multiple sessions.
DVD Digital Versatile Disc; minimum 4.7 GB capacity.
DVD-R DVD Recordable; write once, read many.
DVD-RW DVD-Rewritable; multiple write sessions.

Input Devices


Keyboard


An input device with typewriter-like keys to enter data into a computer.


Layouts:

- QWERTY — Standard; named after first 6 top-row keys (Q W E R T Y).

- DVORAK — Most commonly used letters placed in the middle row.


Key types:


Key Group Description
Alphanumeric Letters, digits (0–9), punctuation, special characters (@, #, &, %)
Modifier keys Shift, Ctrl, Alt — modify other keys when held. E.g., Ctrl+C (copy), Ctrl+V (paste)
Numeric Keypad Right-side calculator pad; 10 digits + 4 operators + Num Lock
Function Keys F1–F12; single-key commands without typing. F1 = Help.
Cursor Movement Arrow keys (↑↓←→), Home, End, Page Up, Page Down
Special Purpose Insert, Delete, PrintScreen, Start key
Ergonomic Keys Designed to reduce hand fatigue; may be curved or split

Wireless keyboards — Use radio frequency (RF) waves. A receiver connects to the computer and receives signals from the keyboard.

Mouse


Controls movement of the cursor/pointer on screen. Invented by Douglas Engelbart at Stanford Research Center in 1963.


Type Description
Mechanical Rolling ball detects movement; older technology.
Optomechanical Ball + optical sensors to detect roller movement.
Optical LED light + sensor tracks surface movement. Most common.
Laser Laser for more precise tracking. Preferred for gaming/design.
Wireless Connects via RF or Bluetooth; reduces desk clutter.

Scanners


Digitizes images by dividing them into a grid of boxes (pixels) — each box represented as 0 or 1 (a bitmap).


- Cannot distinguish text from illustrations — represents all as bitmaps.

- To edit scanned text, OCR (Optical Character Recognition) is needed to convert the image to ASCII characters.


What affects scanner quality:


Factor Description
Scanning Technology Most use CCD (Charge Coupled Device) arrays — rows of light receptors detecting light intensity. Quality of CCD = quality of scanner.
Resolution Denser bitmap = higher resolution.
Bit Depth More bits per pixel = more colors/shades of gray represented.

Types of Scanners:


Type Description
Pen Scanner Scans one line at a time; some can OCR and store text.
Handheld Scanner Scans more than pen scanner; suitable for small pictures.
Flatbed Scanner Scans books, magazines, documents flat on a glass surface.
Sheet-fed Scanner Feeds sheets of paper through the scanner.

Barcode Reader


Reads barcodes — vertical striped marks on product containers.


- Supermarkets use UPC (Universal Product Code).

- Computer links barcode to product description and price, tells POS terminal the price.

- Works like a scanner: shines light on barcode and senses reflection intensity.


Type Description
Handheld Reader Moved against the barcode by user; or fixed while items move over it.
Slot Reader User swipes a card through a slot to decode barcode.

---


Card Readers


Type Description
Business Card Reader Reads and organizes business card information into fields automatically.
Smart Card Reader Reads smart cards with embedded IC (integrated circuit). Used for medical records, identity, access passes.
Credit Card Reader Reads the magstripe (magnetic stripe) on the back of credit cards — made of tiny iron-based magnetic particles in a plastic film.

Digital Camera


Captures photos/videos electronically using a CCD or CMOS sensor that converts light into electronic signals, stored as digital files (usually on SD/microSD cards).


Key features:

- Image Sensor — Core component; determines resolution (in megapixels).

- Lens — Fixed or interchangeable; focuses light onto sensor.

- LCD Screen — Preview and review images/videos.

- Storage — SD or microSD cards.

- Connectivity — Wi-Fi, Bluetooth, USB for easy transfer.

- Modes and Settings — ISO, shutter speed, aperture; scene modes (portrait, landscape, night).


Advantages:

- Instant review and deletion of images.

- Easy digital editing.

- No film cost.

- Easy to store, back up, and share.

Output Devices


Monitors


Most commonly used output device.


1. CRT (Cathode Ray Tube) Monitor

- Electron gun fires beam across screen, lighting phosphor dots.

- Draws lines top to bottom to create full image.

- Monochrome/Grayscale — one electron gun.

- Color (RGB) — three electron guns (Red, Green, Blue); combines intensities for all colors.

- Advantage: Cheap. Disadvantage: Heavy; consumes a lot of power.


2. Flat Panel Monitors


a) LCD — Passive Matrix

- Transistors per row and column create a grid.

- Less expensive. Disadvantage: Narrow viewing angle; poor refresh rate.


b) LCD — Active Matrix (TFT)

- One transistor (actually 4 transistors with TFT) per pixel; each pixel controlled individually.

- Faster refresh; wider viewing angle.

- Disadvantage: More expensive.


c) LED Monitor

- LCD with LED (Light-Emitting Diode) backlighting instead of CCFL.

- Better brightness, contrast, color accuracy.

- More energy-efficient, thinner, lighter, longer lifespan, less heat.


---


Classification of Monitors


By color:


Type Description
Monochrome Two colors — one background, one foreground.
Grayscale Various shades of gray from light to black.
Color (RGB) 16 to 1 million colors; three electron guns (R, G, B).

By signal type:


Type Signal Connections Notes
Digital Binary (0s and 1s) HDMI, DisplayPort, DVI-D, USB-C Sharper image; no conversion noise. Most modern monitors.
Analog Continuous wave (varying voltage) VGA, DVI-A Less sharp; used in older CRTs.

---


Monitor Characteristics


Characteristic Description
Size Measured diagonally in inches.
Resolution Pixel density; more bits per pixel = more colors.
Refresh Rate Times per second screen is redrawn; measured in Hz.
Bandwidth Data transmitted per unit time; digital = bps, analog = Hz.
Interlacing Technique to provide more resolution inexpensively.
Dot Pitch Vertical distance between pixels; lower = crisper image.
Convergence How sharply R, G, B dots converge at each pixel point; misalignment = blurry image.

Printers


Characteristics:


Characteristic Description
Quality Letter quality, near letter quality, or draft quality.
Speed Measured in cps (characters/second) or ppm (pages/minute).
Impact vs Non-impact Impact: strikes ink ribbon (noisy). Non-impact: laser, inkjet (quiet).
Graphics Some (daisy wheel) print text only; others print text + graphics.
Fonts Some limited; most modern printers support multiple fonts.
