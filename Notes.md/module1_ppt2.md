PARTS OF A COMPUTER:
• Central processing Unit(CPU) or processor: The CPU or processor is the brain of  the 
computer. It is the component, that actually executes instructions.
• Memory: Memory enables a computer to store, at least temporarily, data and programs.
• Input device: An input device usually is a keyboard or a mouse. It is channel through which 
data and instructions enter a computer.
• Output device: The typical output devices are display screens(monitors), printers, or other 
such devices that lets you see what computer has accomplished.
• Mass storage device: These devices allow a computer to permanently retain large amount  
of  data. Example: Disk drives, tape drives

CENTRAL PROCESSING UNIT(CPU)
• The part of  the computer that executes program instructions is known as the processor or CPU.
• In a microcomputer the CPU is on a single electronic component, the microprocessor chip within the 
system unit.
The CPU has two main parts
a. Control unit    
b. Arithmetic logic unit(ALU) 
a. Control unit  
• The control unit tells the rest of  the computer system to carry out programs instructions.
• It directs the movement of  signals between memory and ALU.
• It also directs these control signals between CPU and the input/output devices.
• All computer resources are managed from the control unit.
• The CPUs instructions for carrying out commands are built into the control unit.
• Each instruction in the instruction set is expressed in microcode- a series of  basic directions that 
tells the CPU how to execute more complex operations


## Page 5
CENTRAL PROCESSING UNIT(CPU)
The CPU has two main parts
a. Control unit    
b. Arithmetic logic unit(ALU) 
b. Arithmetic logic unit
• ALU performs two types of  operations- arithmetic and logical.
• Arithmetic operations: are mathematical operations like addition, subtraction, 
multiplication etc
• Logical operations: consists of  comparisons. Example: equal to, less than, greater than etc.
• Since data in computer is stored in numbers, processing that takes place inside the 
computer involves carrying out mathematical or logical operations on numbers.
• When control unit encounters an arithmetic or logic operation it passes that instruction 
to the ALU.
• ALU includes a group of  high speed memory locations built directly into the CPU.
• These memory locations are called registers and are used to hold data that is currently 
being processed


## Page 6
Computers have two main types of  memory: primary memory and secondary memory.
Primary Memory
• Primary memory, also known as main memory is directly accessible by the CPU. 
• It is essential for the computer's operation as it stores data and instructions that the CPU needs 
while performing tasks.
 The two primary types of  primary memory are:
1.RAM (Random Access Memory):
1. Function: Temporarily stores data and instructions that the CPU needs during operation.
2. Characteristics: Volatile, meaning it loses its data when the computer is turned off.
3. Types: Includes DRAM (Dynamic RAM) and SRAM (Static RAM)
2.Cache Memory:
1. Function: A smaller, faster type of  volatile memory that stores copies of  frequently accessed 
data from the main memory (RAM).
2. Characteristics: Located closer to the CPU, it significantly speeds up data access times.
3. Levels: Often includes multiple levels (L1, L2, L3) with L1 being the fastest and smallest.
MEMORY
ROM (Read-Only Memory): is considered part of  primary memory. It is non-volatile memory used to store firmware 
and essential system instructions that are required for the computer to boot up and perform basic functions. Unlike 
RAM, which is volatile and used for temporary storage, ROM retains its data even when the computer is turned off.


## Page 7
Secondary Memory
• Secondary memory, also known as auxiliary or non-volatile memory, provides long-term storage 
for data and programs. 
• Unlike primary memory, it retains data even when the computer is turned off.
• Secondary storage/auxiliary storage/external storage
There are two types of  secondary storage devices based on data access: 
a. Sequential access :the data stored in the media can only be read in sequence. To get to a particular 
point on the media you have to go through all the preceding points. Example: magnetic tape
b. Random/Direct access: Random access allows data to be read or written in any order, directly 
accessing any location on the storage medium without having to go through other data sequentially. 
Example: Optical disks, hard disks etc.
MEMORY :Secondary Memory
Advantages of  secondary storage
• Economy: It is more economical to store data on secondary storage devices
• Capacity: More capacity
• Security: Safe from unauthorized people or natural hazards

SCANNERS
• Scanner is an input device that can read text or illustrations 
printed on paper and translate the information into a form 
that the computer can use.
• Scanner works by digitizing an image-dividing it into a grid 
of  boxes and representing each box with either zero or one, 
depending on whether the box is filled in.
• The resulting matrix of  bits is called as a bit map that can be 
stored in a file, displayed on a screen and manipulated by 
programs.
• Optical scanners do not distinguish text from illustrations, they represent all images as bitmaps.
• You cannot directly edit text that has been scanned.
• To edit the text you need an Optical Character recognition(OCR) system to translate the image 
into ASCII characters

MONITORS
• Most commonly used output device.
• There are two types of  monitors
1) CRT monitors    2)Flat panel monitors
1) CRT monitors
• CRT monitors works by moving an electron beam back and forth across the back of  the screen.
• Each time the beam makes a pass across the screen, it lights up phosphor dots on the inside of  
the glass tube, thereby illuminating the active portions of  the screen.
• By drawing many such lines from the top to the bottom of  the screen, it creates an entire 
screen of  images.
• Near the back of  the CRT monitor is an electron gun.
• The gun sends out a beam of  electrons through a magnetic coil, which aims at the front of  the 
monitor.
• In a monochrome and grayscale monitor there is only one electron gun.
• In a color monitor there are three electron guns- one for each primary color(red,green,blue).
• The color monitor displays different colors bycombining various intensities of  three beams.
• Advantage: cheap
• Disadvantage: Heavy, take a lot power

Classification of  monitors-based on signals

CHARACTERISTICS OF MONITORS
1. Size: Screen sizes are measured in diagonal inches, the distance from one corner to the 
opposite corner diagonally.
2. Resolution: Resolution of  a monitor indicates how densely the pixels are packed. The number 
of  bits used to represent each pixel determines how many colours or shades of  gray can be 
displayed.
3. Refresh rate: Refresh rate determines how many times per second the screen is to be 
refreshed(redrawn). Refresh rate is measured in Hertz(Hz)
4. Bandwidth: Bandwidth is the amount of  data that can be transmitted in a fixed amount of  
time. For digital devices, the bandwidth is expressed in bits or bytes per second(bps).For analog 
devices, bandwidth is expressed in cycles per second or Hertz(Hz).
5. Interlaced or non interlaced: Interlacing is a display technique that enables a monitor to 
provide more resolution inexpensively.
6. Dot pinch: Dot pinch is a measurement that indicates the vertical distance between each pixel 
on a display screen. Lower the number crisper the image.
7. Convergence: refers to how sharply an individual color pixel on a monitor appears. Each 
pixel is composed of  three dots- red green and blue. Three dots should converge at the same 
point.If the dots are badly mis converged the pixel will appear blurry.

Laser printer
❖ Laser printers utilizes a laser beam to produce an image on a drum.
❖ The light of  the laser alters the electrical charge on the drum 
wherever it hits.
❖ The drum is then rolled through a reservoir of  toner, which is 
picked up by the charged portions of  the drum.
❖ Finally toner is transferred to the paper through a combination of  heat 
and pressure.
❖ Because an entire page is transmitted to a drum before the toner is 
applied, laser printers are sometimes called page printers.
❖ Monochrome laser printer: uses single toner
❖ Color laser printers: use four toners to print full color.
❖ Non impact printers
❖ Quieter than dot matrix or daisy wheel printers.
❖ Good speed: 4 to 20 pages per minute
